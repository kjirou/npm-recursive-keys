npm-recursive-keys
==================

Object.keys recursively


## Development

### Preparation

- Read [package.json](./package.json).
- Install [PhantomJS](http://phantomjs.org/). e.g. `brew install phantomjs`

### Deployment

```
git clone git@github.com:kjirou/npm-recursive-keys.git
cd ./npm-recursive-keys
npm install
```
